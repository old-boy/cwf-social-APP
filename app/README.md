# app

## Project Created 项目名称：app
```
vue create -p dcloudio/uni-preset-vue app

- [x] 选择 uni-app 默认模板

```

### 安装 uni-app 语法提示
```
npm i @dcloudio/uni-helper-json
npm i @types/uni-app @types/html5plus -D
```

### 启动项目
```
npm run serve
```
